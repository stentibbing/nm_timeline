This is a timeline plugin for Nordic Milk

== Description ==

This adds Timeline custom post type with events as posts. For displaying timeline, nm_timeline shortcode is enabled.

== Installation ==

1. Upload `nm_timeline.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
